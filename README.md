# vis-create-template

一个用于快速创建项目模板的脚手架工具。

## 使用方法

使用 npx（推荐）：
```bash
npx vis-create-template my-project
```

或者全局安装：
```bash
npm install -g vis-create-template
vis-create-template my-project
```

## 可用模板

- template-react-ts: React + TypeScript 模板
- template-vue-ts: Vue + TypeScript 模板

## 注意事项

如果遇到下载模板失败的问题，可以尝试：
1. 检查网络连接
2. 使用 VPN 或代理
3. 稍后重试
4. 或者手动下载模板到本地使用
