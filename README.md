# vis-create-template

一个用于快速创建项目模板的脚手架工具。

## node 版本要求
``` bash
node >= 18.0.0
```

## 使用方法

使用 npx（推荐）：
```bash
npx vis-create-template@latest my-project
```

或者全局安装：
```bash
npm install -g vis-create-template@latest
vis-create-template my-project
```

## 版本说明
当前最新版本：1.0.2

如果您安装的不是最新版本，可以：
1. 使用 `npm install -g vis-create-template@latest` 安装最新版
2. 或清除缓存：`npm cache clean --force`
3. 使用 npx 时指定版本：`npx vis-create-template@latest`

## 可用模板

- template-react-ts: React + TypeScript 模板
- template-vue-ts: Vue + TypeScript 模板
